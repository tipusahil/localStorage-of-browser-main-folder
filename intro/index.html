1. browser er console e application tab e giye (esc) btn press korlei oi application tab er jonno console open hobe, okane js code kora jabe
  2. localStorage er kuno varaible & tar value access korte caile : localStorage.getItem("keyName") evbe likle oi (keyName) varaible er vlaue pawa jave.
  3. localStorage.clear() eta dile localStorage e ja data thakbe sob delete/clear hye jabe.
  4. localStorage e kisu set korte caile varabile/key k quetetion er vtore rakte hobe & value kew quetetion er vitore rakte hbe jmn:
    localStorage.setItem("key/varaible", "value")
    localStorage.setItem("money", 320)
    localStorage.setItem('phone' , "samsumng")
  5. kuno varaible knba key er typeof check korte caile jevabe check korte hbe:
    typeof localStorage.getItem("phone") // string
  6. localStorage theke kuno data/varaible/key remove korte caile :
    localStorage.removeItem("money") // output: money data ta remove hoye jabe.
  7. localStorage e array/object store korte caile seta string akare ortat quetion er vitore rakte kuno 2nd/3rd bracket show korbena.jmn:
    localStorage.setItem("array", [2,3,4,3,4])
    localStorage.getItem("array") //ouptut:'2,3,4,3,4' >evabe string akare dekabe 2nd bracket dekabena
  8. localStorage e object set korte caile: localStorage.setItem( "object1" , {name:"tipu" , age:22} )
